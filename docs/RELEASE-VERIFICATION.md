## Кто проверяет что

Три корзины. «Клод» = автономные харнессы (синтетические клиенты/листенеры на loopback,
декод результата через ffmpeg) — приложения Давида НЕ запускаются, физика не нужна.
Клод-проверка закрывает ЛОГИКУ сценария; строка получает финальный ✅ всё равно только
после живого прогона Давида там, где картинку надо видеть глазами.

**Клод сам (полностью автоматом):**
- A1–A4, A8 — стейт-машина реконнекта relay против фейк-листенера (включая `.waiting`, утечку fd)
- A5 (транспорт) — relay + BlackProgramEncoder → фейк-плагин → ffmpeg-декод → кадры реально чёрные
- B3, B4 — takeover / happy-eyeballs против харнесса BridgeChannelReceiver
- B6 (Bridge-сторона), B7 — auth-handshake синтетическим клиентом: верный/неверный пароль, бан, запрет takeover
- D4, D5 (транспорт) — RTSPOutput-харнесс + ffprobe: формат, ресинк на смене SPS/PPS, чёрный поток
- D6 (транспорт) — SRT-харнесс + ffmpeg-листенер (если ffmpeg собран с SRT)
- F4 (приближение) — синтетический соак харнесса: память/CPU за 40+ мин

**Клоду нужен только запущенный OBS (Давид держит OBS открытым, остальное — Клод):**
- A8 (живой плагин) — флап-тест: синтетический «Bridge» молчит 60 с, линк не рвётся
- A10 — второй захват порта: синтетический листенер на 47788 до OBS → плагин ругается в лог
- F6 — порт занят чужим → лог, не тишина
- E5 (частично) — синтетическая «камера» к прямому источнику, обрыв, реконнект в слот

**Только Давид (физика/глаза/другое железо):**
- A6, A7, A9, A11 — глазами: возврат картинки после чёрного, статус панели, карточка, чужой мак
- B1, B2 (настоящий power-off), B5, B8, B9, B10 — реальный iPhone
- C1–C9 — свитчер глазами + телефонные tally/контролы
- D1, D2 — NDI-приёмник на втором компе; D3 — VLC живьём; D7, D8
- E1–E4, E6–E10 — прямой источник с реальным телефоном
- F1, F2, F3 (соак с реальным AirPlay), F5

## Правила ведения
- Прошёл сценарий → меняем на ✅ Оператор + дата.
- Нашёл баг → сценарий остаётся не-✅, баг чиним, ПЕРЕпроходим сценарий (не верим фиксу без прогона).
- Новая функция → сначала строка здесь, потом код.
